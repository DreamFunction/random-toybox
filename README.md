# random toybox

#### 介绍
写了几个“玩具级”随机小脚本，能造句

#### 说明
随便写的玩具项目，觉得没啥用，但是想留个纪念，再说万一对你有用呢？对于贡献，本作者可能不会回你的pull request（除非你说服我这个项目真的很有用，当然估计没人这么觉得）。玩具也懒得再英文文档了，啥都是中文。文档写的也很随便，不过能看懂就行。


#### 使用说明

都用JSON配。

随机造句程序：分可变（字符串）和不可变词（数组）。造句的时候会随机选一个可变词插到话里。比如：
[["我",["喜欢","不喜欢"],"吃",["苹果","香蕉","梨","桃子"]],["I ",["like","dislike"]," eating ",["apple","banana","pear","peath"]]]
可能输出：我喜欢吃苹果。

随机混词程序：直接用数组，把词一顿瞎混，比如：
["我","看","电视"," I "," watch "," TV "]
可能输出：电视看我
它的前端就是个简易命令行JSON数组制造机。

按语法造句程序：
就做了主谓宾，JSON里多维数组。比如：
[["你","我","他","她"],["打","看","说"],["你","我","他","她"]]
可能输出：你看我
